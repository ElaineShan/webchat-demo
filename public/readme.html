Steps to route a Chat

  1. At the bottom of the page is a list of settings that need to be configured. Use the Auto Fill Org Details button to log into your PureCloud org and have the page automatically fill out the org name and id as well as get a list of queues.
  2. Fill out the Chat end user information or use the auto generated defaults
  3. Click Start Chat to initiate your chat
Org settings will be saved so you won't have to fill them out next time.
